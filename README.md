# Accordion (Vanilla JavaScript)

A simple **Accordion component** built with **HTML, CSS, and Vanilla JavaScript**.

## Features
- Expand and collapse accordion items
- Only one item opens at a time
- Click-based toggle logic
- Clean and minimal implementation
- Beginner-friendly project

## Description
This project demonstrates an accordion pattern where clicking on a header reveals its content while closing the others. The logic is handled using JavaScript conditions and DOM manipulation to manage active states.
